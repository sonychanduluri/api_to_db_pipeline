# API to Database Data Pipeline (ETL Project)

## 📌 Project Overview
This project is an automated **ETL (Extract, Transform, Load) data pipeline** built using Python.  
It extracts real-time cryptocurrency market data from the **CoinGecko REST API**, transforms the raw JSON data into a structured format, and loads it into a **SQLite database**.

This project demonstrates real-world skills like API integration, data transformation, and database storage.

---

## 🚀 Features
- Extracts real-time data from CoinGecko REST API
- Transforms raw JSON into a clean structured DataFrame using Pandas
- Loads transformed data into SQLite database
- Modular design (Extract, Transform, Load)
- Stores data in a database table for analytics and querying

---

## 🛠 Tech Stack
- Python
- REST API (CoinGecko)
- Requests
- Pandas
- SQLAlchemy
- SQLite
